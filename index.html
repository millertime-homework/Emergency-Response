    Frequently Asked Questions

    In the earthquake scenario there are some objectives that I failed and I'm not sure why.

    There are a lot of optional objectives in the earthquake scenario that are only possible before the earthquake happens. They don't cause you to get a game over; they are only there for bonus points. If you want to play again to try to find them, this is highly encouraged. Any objectives that aren't completed - optional or not - are reviewed at the end of this scenario. This is done using the objective system.

    If we didn't use the objective system to track these optional objectives, then there would be no "failed objectives" when the optional ones aren't completed. However, this would affect the ending sequence that explains which parts were missed.

    I'd like to add a feature to the objective system that marks certain objectives as being optional, this way they aren't "failed" when they aren't completed. When the objectives are listed at the end, they wouldn't be there at all.

    At this time, there is no ETA on completing these changes, so unfortunately there will continue to be some overly false information at the end of the earthquake scenario.

    I am stuck at the smoky hallway.

    This was a frustrating point to reach, and it wasn't extremely clear that the smoke needed to be clicked. The smoke can still be clicked, but now you can also click your breathing mask in your backpack (if you grabbed it at the beginning) -- or you can pass through unharmed by using the water leaking down the wall nearby to wet your shirt.

    The tutorial isn't clear enough and needs more content.

    The tutorial does need more content. Here are some items that should show up in the near-to-distant future.

    • beginning text needs to be reworded
    • ending needs to be less abrupt
    • blue glow needs to be pointed out (we put this on all clickable items)
    • a character needs to be in the tutorial that the player is told to talk to
    • a door needs to be added that the player is told to click to open
    • the "Help" button needs to be pointed out and its purpose explained
    • the point system needs to be explained

    I'm using Internet Explorer and the game doesn't seem to work right.

    We have done a lot to resolve incompatibilities in our code with Internet Explorer. There are still some existing intermittent issues that aren't fully resolved, and we recommend playing the game in Google Chrome or Firefox.

    The points system doesn't make sense. My score is ridiculous.

    The points awarded for the objectives in the game are currently very arbitrary. Sadly no tuning has been done to make the points sensical. So sometimes you get a lot of points very easily and sometimes you hardly get any.

    This is a somewhat large task that needs to be taken on to vastly improve the experience of the game, but unfortunately there is no ETA for when it will be completed.
